# Tailwind

- Core concept: Class base
- Mỗi thuộc tính css và 1 giá trị css --> tạo ra class riêng
  Ví dụ: text-xl --> font-size

## Cú pháp

```
tiento-giatri --> giá trị do tailwind build sẵn
tiento-[giatri] --> giá trị custom
pseudo:tiento-giatri
pseudo:tiento-[giatri]
breakpoint:tiento-giatri
breakpoint:tiento-[giatri]
```

Group

```
group-pseudo:tiento-giatri
group-pseudo:tiento-[giatri]
```

Nested Group

```
group-pseudo/group-name:tiento-giatri
group-pseudo/group-name:tiento-[giatri]
```
