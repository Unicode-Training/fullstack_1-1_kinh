## Bài 2:

Viết một hàm `createCalculator(initialValue)`. Hàm này khởi tạo một giá trị ban đầu và trả về một hàm con `operate(number, callback)`.

Mỗi khi operate được gọi:

- Nếu number nhỏ hơn 0, giá trị trong hàm `createCalculator` không thay đổi, và gọi callback với tham số là -1.

- Nếu number lớn hơn hoặc bằng 0, cộng number vào giá trị hiện tại của hàm `createCalculator`, và truyền giá trị mới này vào callback để xử lý tiếp, sau đó trả về kết quả của callback.

```js
const createCalculator = (initialValue) => {
  //Code tại đây
};
```

Test:

```js
const myCalc = createCalculator(10);

console.log(
  myCalc(5, (res) => {
    return res * 2;
  }),
); //Kết quả: 30 ((10 + 5) * 2)

console.log(
  myCalc(-3, (res) => {
    return res + 10;
  }),
); //Kết quả: 9 (-1 + 10)

console.log(
  myCalc(2, (res) => {
    return res;
  }),
); //Kết quả: 17 (15 + 2)
```


Test

```js

});
```
