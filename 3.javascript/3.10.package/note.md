# Các lệnh npm

1. Khởi tạo

npm init -y

2. Cài đặt thư viện

npm i ten-thu-vien
npm i ten-thu-vien@phienban

3. Cài đặt tất cả thư viện có sẵn trong package.json

npm i

3. Cài đặt thư viện chế độ dev

npm i tenthuvien --save-dev
npm i tenthuvien@phienban --save-dev

4. Chỉ cài đặt tất cả thư viện ở chế độ dependency

npm i --production

5. Gỡ bỏ thư viện

npm un tenthuvien

6. Update thư viện lên phiên bản mới

npm update tenthuvien

7. Update tất cả thư viện

npm update
