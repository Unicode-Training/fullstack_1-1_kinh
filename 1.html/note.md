# HTML

## Tổng quan

- Ngôn ngữ đánh dấu siêu văn bản
- Cấu hình bởi các thẻ
  Ví dụ:

  ```html
  <tenthe thuoctinh1="giatri1" thuoctinh2="giatri2">Noi dung</tenthe> --> Thẻ
  đầy đủ
  ```

  ```html
  <tenthe thuoctinh1="giatri1" thuoctinh2="giatri2" /> --> Thẻ tự đóng, thẻ rỗng
  ```

- Nhóm thẻ:

* inline

- chiều rộng mặc định bằng với nội dung
- Các thẻ inline đứng cạnh nhau tạo thành 1 hàng

* block

- Mỗi thẻ ở 1 hàng riêng biệt
- Chiều rộng mặc định bằng với chiều rộng của thẻ cha

## Điều gì khó nhất khi học HTML

- Semantic: Ngữ nghĩa
- Code dễ đọc
- Bảo trì, nâng cấp, làm việc team
- Phát triển web tương thích với những đối user là người khuyết tật
